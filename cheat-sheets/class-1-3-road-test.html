Class 1/3 Road Test Cheat Sheet

Test Day Reference · BC + Alberta · Print-friendly

Pre-Trip Cheat Sheet →

What to Bring on Test Day

  • Valid ID (driver's licence + photo ID if asked)
  • Learner's permit / Class 7 with appropriate endorsement
  • Medical certificate (if required by province)
  • The truck — clean, fueled, working, no major defects
  • All paperwork: ownership/registration, insurance, CVIP if commercial
  • Hi-vis vest, steel-toe boots, gloves
  • Logbook (if required)
  • Tire-thumper, chock blocks, fire extinguisher in cab

Test Sequence — What Examiners Watch

  1. Pre-trip walk-around inspection — verbal + physical check (E-C-L-M-A-B method)
  2. Air brake test sequence — static, applied, low-air warning, spring brakes, parking
  3. Coupling check (Class 1 only) — fifth wheel, kingpin, air lines, electrical, tug test
  4. In-cab familiarization — controls, mirrors, seatbelt, gauges
  5. Backing maneuver — straight back / offset / alley dock (depends on facility)
  6. Road portion — city streets, highway, lane changes, intersections, hills
  7. Shutdown sequence — park properly, set brakes, secure, walk-around if requested

Top 10 Reasons Drivers Fail

  1. Skipping items in pre-trip — missed lights, missed under-truck check
  2. Wrong air brake test sequence — out of order or skipping steps
  3. Pushrod stroke not measured or measured wrong
  4. Not identifying defects correctly (major vs minor confusion)
  5. Rolling stops — failing to come to complete stop
  6. Improper lane change — no shoulder check, wrong signal timing
  7. Hitting curbs while turning — wide-turn miscalculation
  8. Speed control — too fast in city, too slow on highway
  9. Backing without GOAL (Get Out And Look)
  10. Improper shutdown — leaving cab without parking brake set

Backing Maneuver — Key Points

  • GOAL: Get Out And Look. Always allowed, never penalized.
  • Mirror checks every 2 seconds
  • Use 4-way flashers when backing
  • Counter-steer early, not late — trailer reacts slow
  • Don't be afraid to pull forward and reset
  • Speed: walking pace or slower

Mirror & Shoulder Check Routine

This pattern is what examiners watch for at every lane change, intersection, and turn:

MIRROR · SIGNAL · MIRROR · SHOULDER · MOVE
  • Lane change: Side mirror → signal 3+ seconds → side mirror → shoulder check → move
  • Right turn: Right mirror frequently — watch for cyclists/peds
  • Left turn: Left mirror, scan for traffic both directions twice
  • Intersection: Look both ways even on green light

Hill Starts & Downhill

Uphill Start

  1. Apply parking brake while stopped
  2. Engage low gear, build clutch engagement
  3. As clutch grabs, release parking brake smoothly
  4. No rollback — examiner watches

Downhill Driving

  • Snub braking: firm app to drop ~5 mph below safe speed, release, repeat
  • Use engine braking / Jake brake / lower gear
  • Never coast in neutral — illegal and unsafe
  • Choose your gear BEFORE the hill, not on it

Shutdown Sequence (Order Matters)

  1. Pull straight, fully off road if applicable
  2. Apply service brake, come to complete stop
  3. Shift to neutral
  4. Apply parking brake (yellow knob)
  5. Apply trailer brake (red knob, if combo)
  6. Idle 3+ minutes if turbo (cool down)
  7. Engine off, key out, secure
  8. Chock wheels if grade requires

10-Second Pre-Test Reminder

  • Walk-around BEFORE climbing in
  • Air test in correct order — static then applied
  • Pushrod stroke under spec (Type 30: ~1.5")
  • GOAL while backing — Get Out And Look
  • Mirror-Signal-Mirror-Shoulder-Move on every lane change
  • No rolling stops, even at empty intersections
  • Shutdown sequence in order, every time
